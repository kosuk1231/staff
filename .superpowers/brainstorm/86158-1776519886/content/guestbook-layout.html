A — 하트 격자형

각 메시지가 하트 모양 버튼. 이름이 하트 안에 표시되고, 클릭하면 메시지 팝업. 화면을 가득 채우는 하트 그리드.

전체가 하트 실루엣 → 각 점이 메시지 하트

B — 하트 실루엣 배치형

현재 방식 개선. 전체 배치가 큰 하트 모양을 이루고, 각 포인트가 다양한 크기의 하트 아이콘. 클릭 시 메시지 팝업.

카드가 쌓이는 형태 (세로 스크롤)
김민지
40주년 축하합니다! 🎉
이서연
서울 복지의 미래를 응원합니다!
박지우
항상 함께하겠습니다 💪

C — 하트 카드 리스트형

각 메시지가 하트 아이콘 + 이름 + 메시지 미리보기로 구성된 카드. 새 메시지가 위로 쌓임. 클릭 시 전체 메시지 확인.

💡 A+B 조합도 가능합니다 — 상단에 하트 배치 시각화, 하단에 리스트
